# racing-horse (Vite + React + styled-components + Axios) — JavaScript 버전

## 기술 스택
- React 18
- Vite 5
- JavaScript (TypeScript 미사용)
- styled-components 6
- Axios 1

## 실행 방법
1) 의존성 설치
```powershell
cd client-vite
npm install
```

2) 개발 서버 실행
```powershell
npm run dev
```

3) 프로덕션 빌드 및 미리보기
```powershell
npm run build
npm run preview
```
